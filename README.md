# Rat Tracker

Single-user workout-tracking PWA for desktop and mobile.

## Development

```powershell
npm.cmd install
npm.cmd run dev
```

## Local Network Preview

```powershell
npm.cmd run build
npm.cmd run preview:host -- --port 4173
```

## Checks

```powershell
npm.cmd run lint
npm.cmd run check:data
npm.cmd run build
```
