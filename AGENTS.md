# Gym Tracker PWA Instructions

## Project purpose
This is a single-user workout-tracking PWA designed for desktop and mobile use.

## Technology
- Vite
- React
- TypeScript
- Dexie / IndexedDB

## Development rules
- Inspect existing types and components before changing architecture.
- Prefer small, testable changes.
- Preserve existing Dexie data unless a migration is necessary.
- Do not rename files or move folders unless required.
- Run the project after changes and report any TypeScript or runtime errors.
- Explain which files changed and why.
- Do not mark a feature complete until the user confirms it works.
